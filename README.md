# CSI22-Catalogo-de-Prestadores-de-ServiÇos-em-Tkinter

Projeto de elaborar um catálogo de prestadores, de serviços. 
Com um banco de dados com seus dados, permitir cadastro, consulta e deleção. Primeiramente utilizando-se a biblioteca Tkinter de elaboração gráfica do python.
Paralelamente, considerar a refatoração do código anterior realizado.
Projeto desenvolvido para a disciplina de orientação a objetos pelos alunos Ricardo Cardoso, Luiz Satoshi e Leonardo Dantas.
